# Milestone v1.0: devproxy

**Status:** Shipped 2026-05-12
**Phases:** 1-4
**Total Plans:** 25

## Overview

`v1.0` delivered the first complete devproxy release for macOS local Docker Compose workflows. The milestone established deterministic route discovery, local DNS plus HTTP/HTTPS serving, macOS install and diagnostics, and the menu bar/dashboard operator surfaces on top of the shared admin API.

## Milestone Summary

- Deterministic routing now derives from live Docker Compose metadata with explicit conflict and warning visibility.
- Managed `.test` DNS plus local HTTP/HTTPS listeners now serve active routes with mkcert-backed certificates and friendly fallback behavior.
- macOS install, uninstall, doctor, status, logs, and refresh flows are wired through one daemon-owned admin API.
- Menu bar and dashboard surfaces now expose live daemon state and route actions without bypassing the shared control plane.
