# Phase 4 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (manual contracts) |
| Preset | not applicable |
| Component library | none (native macOS menu bar + server-rendered dashboard HTML) |
| Icon library | SF Symbols for menu bar status glyphs; inline SVG Feather-style icons for dashboard cards/actions |
| Font | macOS system stack (`-apple-system`, `BlinkMacSystemFont`, `Segoe UI`, `sans-serif`) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: menu-bar click targets and toggle rows use minimum 44px height for pointer accessibility.

|------|------|--------|-------------|

| Label | 12px | 600 | 1.4 |
| Heading | 20px | 600 | 1.2 |

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0F172A | App background, menu panel background, dashboard shell |
| Secondary (30%) | #1E293B | Cards, grouped sections, side nav, route rows |
| Accent (10%) | #22C55E | Status “Healthy”, primary CTA, active toggle state, selected route row |
| Destructive | #DC2626 | Stop/disable confirmations and hard failure chips only |

Accent reserved for: pause/resume primary action, dashboard “Refresh Routes” CTA, healthy-status indicator, active start-at-login toggle state, focused route-open action.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Refresh Routes |
| Empty state heading | No Active Routes |
| Empty state body | DevProxy did not find routable containers. Start your Compose project, then select **Refresh Routes**. |
| Error state | DevProxy can’t reach the daemon right now. Ensure the daemon is running, then select **Run Doctor** for repair guidance. |
| Destructive confirmation | Pause routing: “Pause routing for all managed domains? DNS will still resolve, but requests will return a paused response until resumed.”; Disable start at login: “Turn off start at login for this role? You’ll need to launch it manually next session.” |
